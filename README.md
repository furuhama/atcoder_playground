## AtCoder Playground

Run sample codes on AtCoder equivalent environment

## Languages

- Rust (1.42.0 with all available libraries)
- Ruby (2.7.1)
- C++ (gcc 9.2.1)

## Requirements

- cargo
- rbenv
- gcc (9.2)

## Usage

```
$ make <language-name>

# examples

$ make rust
$ make ruby
$ make cpp
```
